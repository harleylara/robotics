## Prerequisites for Diving

Knowledge required:
- Familiarity with Ubuntu’s desktop environment. I recommend the book @blumLinuxCommandLine2025 (Edition 3rd/4th or 5th) chapter 1, section "The Linux Desktop Environment".
- Proficiency with basic terminal commands (mainly bash). See chapter 3 and 4 of @blumLinuxCommandLine2025. [Linux Cheat Sheet](/en/ros2/linux-cheat-sheet).
- Understand GNU/Linux file system. @blumLinuxCommandLine2025 chapter 7 and 8.
- Python/C++ skills, including basic [Object-oriented programming (OOP)](https://en.wikipedia.org/wiki/Object-oriented_programming) concepts. For recap python check [Think Python 3rd Edition by Allen B. Downey](https://greenteapress.com/wp/think-python-3rd-edition/) for C++ check @stroustrupProgrammingPrinciplesPractice2024
- Basics of [Version control system (VCS)](https://en.wikipedia.org/wiki/Version_control) with git. My personal recommendation is the [Pro Git 2nd Edition by Scott Chacon and Ben Straub](https://git-scm.com/book/en/v2) chapter 1, 2, and 3. [Git Cheat Sheet](https://git-scm.com/cheat-sheet).

software requirements:
- OS: Ubuntu 22.04 Desktop. I personally recommend a native Ubuntu installation on your computers. Dual-boot also works but very impractical for daily use (in my opinion). If you want to use a virtual machine I recommend using VMware Workstation Pro it's free for personal non-commercial use (runs on GNU/Linux and Windows).
- Install: ROS Humble, check the official documentation and follow [Ubuntu Debian packages (recommended)](https://docs.ros.org/en/humble/Installation/Ubuntu-Install-Debians.html) to install it.

Sometimes I get asked for resources on more general robotics topics or ROS-specific material—if that's what you're looking for, check my personal [Bookmarks – Robotics](https://bookmarks.harleylara.com/subjects/#robotics).

**Disclaimer**: these notes aren’t meant to be a go-to resource for learning the Linux terminal, bash scripting, programming (Python or C++), Git, or general robotics topics like transformation matrices and similar. The resources mentioned above are significantly more complete and better suited than what I can cover here, so I highly recommend checking them (or any other resource you prefer) if you feel like you’re lacking in those areas or just want to reinforce your existing knowledge.

**Expectation for these notes**: the main goal of these notes is understanding the mechanisms that make up ROS and the ecosystem around it, while building knowledge incrementally and in a logical progression of topics that would (hopefully) help you understand the concepts with ease.
