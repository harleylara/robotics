## Outline

- Core and common sensors:
    - gyroscope
    - accelerometer
    - magnetometer (compass)
    - barometer
    - Airspeed sensor (for fix-winds)
    - GPS. See [Big GPS Round Up - Ardupilot Blog](https://discuss.ardupilot.org/t/big-gps-round-up/67755) by Andrew Tridgell
