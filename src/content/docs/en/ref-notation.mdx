## Numbers and Arrays

| Description | Representation |
| --- | ---|
| Scalar | $a$ |
| Vector | $\va$ |
| Matrix | $\mA$ |
| Tensor | $\tA$ |

## Set and Graphs

| Description | Representation |
| --- | --- |
| Set | $\sA$ |
| Set of real numbers | $\sR$ |
| Set containing $0$ and $1$ | $\{0, 1\}$ |
| Set of all integers between $0$ and $n$ | $\{0, 1, \dots, n\}$ |
| Real interval including $a$ and $b$ | $[a, b]$ |
| Real interval excluding $a$ but including $b$ | $(a, b]$ |
