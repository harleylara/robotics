## The Jungle of Formats

There are some many formats to represent robots (and other things):
- [URDF (Unified Robot Description Format)](https://wiki.ros.org/urdf)
    - [URDF improvements](https://discourse.ros.org/t/urdf-improvements/30520)
    - [Short-term WG about URDF / description formats](https://discourse.ros.org/t/short-term-wg-about-urdf-description-formats/30843)
    - [xacro](https://wiki.ros.org/xacro) in [Github](https://github.com/ros/xacro)
- [SDF](http://sdformat.org/)
    - [SDFormat extensions to URDF](http://sdformat.org/tutorials?tut=sdformat_urdf_extensions&cat=specification&)
- [Mujoco XML](https://mujoco.readthedocs.io/en/stable/XMLreference.html#introduction)
- [Universal Scene Description (USD) or OpenUSD](https://openusd.org/release/index.html)
