## Create a Custom Service defintion

create `.srv` file


```python filename="EuclideanDistance.srv" remoteURL="https://raw.githubusercontent.com/harleylara/ros2-intro/master/demo_interfaces/srv/EuclideanDistance.srv"
```

Add dependencies and append `.srv` to `rosidl_generate_interfaces`

```cmake
...

find_package(rosidl_default_generators REQUIRED)
find_package(geometry_msgs REQUIRED)

rosidl_generate_interfaces(${PROJECT_NAME}
  "msg/Counter.msg"
  "srv/EuclideanDistance.srv"
  DEPENDENCIES geometry_msgs
)

...
```

Add any dependencies to `package.xml`

```cmake
    <build_depend>geometry_msgs</build_depend>
```
