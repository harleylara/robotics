If we consider each measurement to be equally uncertain, we can sum the square of all errors together, for all measurement points, to quantify an overall fit between the line and all of the measurements:

<Equation
    formula="S = \underset{i}{\sum} d_i^2"
    description="Overall error"
/>

Our goal is to minimize $S$ when selecting the line parameters $(\alpha, r)$. We can do so by solving the nonlinear equation system:

<Equation
    formula="\frac{\partial S}{\partial \alpha} = 0 \> \> \frac{\partial S}{\partial r} = 0"
    description="Minimize error"
/>
