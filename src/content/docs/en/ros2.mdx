## Prerequisites for Diving

Knowledge required: 
- [Linux Command Line and Shell Scripting Bible, 4th Edition](https://www.wiley.com/en-us/Linux+Command+Line+and+Shell+Scripting+Bible%2C+4th+Edition-p-9781119700937)
    - Familiarity with Ubuntu's desktop environment
    - Understand GNU/Linux file system
    - Proficiency with basic terminal commands (bash)
- [Python Data Science Handbook by Jake VanderPlas](https://jakevdp.github.io/PythonDataScienceHandbook/)
    - Python/C++ skills, including basic OOP concepts
- [Pro Git by Scott Chacon and Ben Straub](https://git-scm.com/book/en/v2)
    - Basics of VCS (Version control system) with git

Software requirements:
- OS: Ubuntu 22.04 Desktop
- Install: ROS Humble, check the official documentation and follow [Ubuntu Debian packages (recommended)](https://docs.ros.org/en/humble/Installation/Ubuntu-Install-Debians.html)
