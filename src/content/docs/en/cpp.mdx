## Prerequisites for Diving

Knowledge required:
- Proficiency with basic terminal commands (bash).
- Understand GNU/Linux file system.
- I expect that you already know the general concepts of programming, including interpreted vs. compiled.
- Basics of VCS (Version control system) with git.

Software requirements:
- Text editor: [Visual Studio Code](https://code.visualstudio.com/) with the [C/C++ by Microsoft](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools) extension.
- Compiler: `gcc`, a reference article for installation, [Using GCC with MinGW](https://code.visualstudio.com/docs/cpp/config-mingw#_prerequisites).
