## GLSL

GLSL view a programming language is very similar to `C`

Types:
- `int`
- `float`
- `bool`

Including:
- `vec2`, `vec3`, `vec4`
- `mat2`, `mat3`, `mat4`

## Transformation Pipeline

- `Local space` is your own frame of reference
- `World space`
- `View space`
- `clip space`, from here OpenGL take care of everthing `clip space` -> [`NDC` (normalized device coordinate)](https://learnopengl.com/Getting-started/Coordinate-Systems) -> `screen space`


Notes:
- `uniform` is the same in the vertex Shader and the fragment shader, this is like a global parameter.
- `attributes` defined in each vertex. Usefull for per-vertex parameters
- `varyings` like a communication pipe
