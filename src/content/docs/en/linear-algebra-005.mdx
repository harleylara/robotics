## Vector Spaces

space: refers to a set (space) of vectors.

Vector space requirements, the sum of any vector $\vv + \vw$ and the multiplication $c\vv$ **stay in the space**.

In other words: All linear combinations of $c\vv + d\vw$ are **in the space**.


Example:
- $\sR^{2}$: all posible 2-dim **real** column vectors. Represent a plane.
- $\sR^{n}$: all posible n-dim **real** column vectors.

<Important
    note="Definition: The space $\sR^n$ consists of all column vectors $\vv$ with $n$ components (the components are real numbers)."
/>
